# Website Checker Automation Tool

This project uses Python and Selenium to check if websites are loading correctly and reports pass or fail results.

## Tools Used
- Python
- Selenium
- ChromeDriver

## How to Run
1. Install dependencies:
   pip install selenium

2. Run the script:
   python test_websites.py

## Project Structure
- tests/ → test scripts
- data/ → list of websites
- utils/ → driver setup
- reports/ → output files
